[routerLink] Navegação sem recarregar a página: O routerLink permite que você navegue entre as rotas do aplicativo sem recarregar toda a página. Isso é essencial em aplicativos de página única (SPA) para uma experiência de usuário mais suave, pois apenas as partes relevantes da página são atualizadas.
